Day 75 - JavaScript: Events - Pt. 4

Another advantage of using event listeners is that the callback function receives an `event` variable as an argument. This variable contains the information about the event itself, for example, the kind of event (e.g., click, key press, change).

In click events, this variable also contains the location of the click:

const loc = document.querySelector('#location');
const button = document.querySelector('button');

button.addEventListener('click', logClick);

function logClick(e){
 // e is a reference to the event itself
 loc.textContent = `Location - x: ${e.x}, y: ${e.y}`;
}

Furthermore, you can access the element that triggered the event to manipulate it:

function logClick(e){
 ...
 counter += 1;
 // access the button throughout the event variable
 e.srcElement.textContent = `Clicked ${counter} times`;
}
