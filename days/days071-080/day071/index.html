Day 72 - JavaScript: Events - Pt. 1

Now that we have learned how to manipulate the DOM with JavaScript, it's time to learn how to trigger these changes dynamically using events. Events are actions on the webpage, such as clicks or key presses.

There are several ways to "listen" to events. The first and simpler one is to specify function attributes on the HTML elements.

Suppose you have a button on your webpage, and you want to increment a counter every time the user clicks the button. First, define the JavaScript function that will make the processing:

let counter = 0; // Variable to store the number of clicks
const counterField = document.querySelector('#counter'); // Reference to the element that shows the count.

function logClick(){
 counter += 1; // One new click
 counterField.textContent = `Clicks count: ${counter}`; // Update the text
}

There is a function called `logClick` in the script. This function will be called every time the button is pressed. To bind the function and the button, use the button's onclick attribute specifying the function to be executed as the argument:

<button onclick="logClick()">Click me!</button>

Finally, add the element that will show the counter:

<p id="counter">Clicks count: 0</p>

Clicks count: 0
