Day 76 - JavaScript: Events - Pt. 5

Yet another advantage of using event listeners is the ease of attaching listeners to a group of elements.

Imagine that you have 26 buttons, one for each letter of the alphabet, and you want to display which button was pressed in a text field. Setting the onclick attribute on each button in the DOM would be a burden, but this task is easy with JavaScript.

To attach an event listener to a group of elements, use the forEach() method:

const buttons = document.querySelectorAll('button'); // All the buttons
const display = document.querySelector('#display');

// Loop to each button in the group and attach the listener to each one
buttons.forEach(button => button.addEventListener("click", logClick));

function logClick(){
 // `this` refers to the button in this context
 display.textContent = `You have pressed button ${this.textContent}`;
}

Note the use of the arrow function, which is equivalent to:

buttons.forEach(clickButton);

function clickButton(button){
 button.addEventListener('click', logClick);
}

For brevity, arrow functions are preferred in this type of application.

No button has been clicked
