Day 73 - JavaScript: Events - Pt. 2

The second approach to handling events is, instead of cluttering the HTML document setting the onclick attribute on the DOM element, to use JavaScript to set the onclick property.

In this way, the HTML document is straightforward:

<button>Click me!</button>
<p id="counter">Clicks count: 0</p>

And the script does all the work:

let counter = 0; // Variable to store the number of clicks

const counterField = document.querySelector('#counter');
const button = document.querySelector('button'); // Reference to the button

button.onclick = logClick; // Bind the function to the onclick event

function logClick(){
 counter += 1; // One new click
 counterField.textContent = `Clicks count: ${counter}`; // Update the text
}

This approach is preferred over setting the onclick attribute on the DOM element, as it separates the concerns of design and operation.
