Day 74 - JavaScript: Events - Pt. 3

The third approach to handling events is to attach an event listener to the element using JavaScript. This approach is similar to setting the element's property on[event], with a main difference: you can add several listeners (i.e., several functions) to the same event.

Suppose that, besides counting the clicks, you want to move the button every time it is pressed. You can accomplish this task using two different functions, one that counts the clicks and the other that pushes the button.

let counter = 0; // Variable to store the number of clicks

const counterField = document.querySelector('#counter');
const button = document.querySelector('button');

// Attach both listeners
button.addEventListener('click', countClick)
button.addEventListener('click', incrementMarginLeft)

function countClick(){
 counter += 1; // One new click
 counterField.textContent = `Clicks count: ${counter}`; // Update the text
}

function incrementMarginLeft(){
 button.style.marginLeft = counter * 10 + "px"; // Move the button every click
}

The HTML is straightforward:

<button>Click me!</button>
<p id="counter">Clicks count: 0</p>

Now, the button has two listeners attached to click events, one for counting the clicks and the other for moving the button to the right side 10 pixels every click.
