Day 71 - JavaScript: DOM Manipulation, Pt. 7

Element and attribute removal

JavaScript also allows us to remove elements of the DOM. To perform this action, first target both the parent and the child (the element to be removed) and use the parentNode.removeChild() method.

Suppose you have the following HTML document:

<body> <h1 style="text-align: center;">The title</h1>
<p class="red">This is the first paragraph.</p>
<h2>The subtitle.</h2>
<p>This is the second paragraph.</p>
</body>

Which should render as:

The title

This is the first paragraph.

The subtitle.

This is the second paragraph.

You can remove the last two elements (the subtitle and the second paragraph) using the following script:

const body = document.querySelector('body');

const p2 = body.lastElementChild;
const h2 = p2.previousElementSibling;

body.removeChild(p2);
body.removeChild(h2);

Also, the attributes (e.g., ID, class, style) can be removed using the node.removeAttribute() method. To clear the style of both the title and the first paragraph, use the following code:

const h1 = body.firstElementChild;
const p = h1.nextElementSibling;

h1.removeAttribute('style');
p.removeAttribute('class');

The final result should be rendered as:

The title

This is the first paragraph.
