(1) Normal Flow

(2) The standard way to allocate elements on the page is called the (3) Normal Flow, which is (4) top to bottom and left to right (except for languages like Hebrew or Arabic).

(5) Block elements will be placed one below the other, and inline elements will be placed on the right if there's enough space or below another if there isn't enough space.

(6) All the elements on this page are numbered and should appear in the order declared in the code.

(7) There are several ways to alter the normal flow, one of which is using the (8) position property. For example, (9) this block will appear way below its original position.

(10) By only using the Normal Flow, only very basic (and not very appealing) web pages can be made. Altering the normal flow allows us to create complex and more attractive designs.
