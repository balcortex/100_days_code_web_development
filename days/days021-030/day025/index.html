Flexbox Shrinking

Another property of the flexbox layout, apart from growing, is the ability to shrink the items to fit the container. The CSS property that controls the shrinking is flex-shrink, a property of the flex-items. Let's explore an example.

Suppose you have a flex-container with three flex-items, each with a fixed width of 220px. The default behavior of the flex-items is to shrink to fit the container width. That is, the default value of the flex-shrink property is 1:

Item 1 (shrink: 1)
Item 2 (shrink: 1)
Item 3 (shrink: 1)

If we set the value 0 to all the flex-items, these items are not allowed to shrink. In this example, the items cannot fit into the container, and overflows occur.

Item 1 (shrink: 0)
Item 2 (shrink: 0)
Item 3 (shrink: 0)

Finally, we can combine items that don't shrink with items that shrink more (setting a value greater than 1) or shrink less (assigning a value between 0 and 1):

Item 1 (shrink: 0)
Item 2 (shrink: 2)
Item 3 (shrink: 0.5)

Note: negatives values are invalid.
