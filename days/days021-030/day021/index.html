The Inline-Block Display Value

First, let's remember the fundamental values of the display property:

  • Block: sets the element in a new line
    display: block
    display: block
  • Inline: puts one element next to another element. Both must share the inline value.
    display: inline
    display: inline

If you look at the page code, both the block and inline elements must be 230px wide and 40px high, but only the block element follows these properties. The inline element always wraps the content, which means its width and height cannot be specified with the width and height properties.

Now, let's introduce the inline-block value, which behaves like an inline element, but their size can be adjusted via the width and height properties:

display: inline-block
display: inline
display: inline-block
