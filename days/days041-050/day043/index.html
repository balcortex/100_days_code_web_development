JavaScript

JavaScript is a scripting or programming language that allows you to implement complex features on web pages, like displaying timely content updates, interactive maps, animated graphics.

It is the third layer, HTML and CSS being the other two, of the layer cake of standard web technologies. The three layers build on top of one another nicely.

JavaScript was initially created to “make web pages alive”.

The programs in this language are called scripts. They can be written right in a web page's HTML and run automatically as the page loads. They don't need special preparation or compilation to run. In this aspect, JavaScript is very different from another language called Java.

(When JavaScript was created, it initially had another name: “LiveScript”. But Java was very popular at that time, so it was decided that positioning a new language as a “younger brother” of Java would help.)

What can JavaScript do?

  • Change the existing content, modify styles.
  • React to user actions, run on mouse clicks, pointer movements, key presses.
  • Send requests, download and upload files.
  • Ask questions to the visitor, show messages.
