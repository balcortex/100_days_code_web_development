JavaScript Conditionals: The Switch Statement

Although the IF statement is generally the to-go option, the SWITCH statement is clearer (and faster) in situations where many conditions are provided.

For example, consider that the user has the option to choose between one of the following themes:

  • Light
  • Dark
  • Blue
  • Sunset
  • Dusk
  • Springtime

You could implement the conditional with a series of nested IF statements. However, with SWITCH the code is much clear.
