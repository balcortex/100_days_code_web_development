JavaScript: Variables

Variables are essential to any program. They allow us to store information: numbers, letters, arrays, and many others.

The most common and flexible way to define a variable is by using the keyword let:

let myVar = 0

This line of code declares a variable called myVar and assigns the numerical value of 0 to it.

To illustrate this, let's make a function which increments the value of the variable each time a button is pressed:

<script>
 let counter = 0
 let text = document.querySelector("h4");

 text.textContent = `You have pressed the button ${counter} times`;

 function increment() {
  counter = counter + 1
  text.textContent = `You have pressed the button ${counter} times`;
 }
</script>
