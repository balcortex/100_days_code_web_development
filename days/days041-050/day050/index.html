JavaScript: Advanced Logging

We can display helpful information for ourselves (or other developers) using the console.log() function:

console.log("This is a simple string on the console");

Also, we can include the value of any variable, even more, an operation, using the string backtick syntax:

let x = 2;
let y = 3;
console.log(`x=${x}, y=${y}, and x+y=${x+y}`);

Other functions, besides log, display information on the console, each with a different style to establish visual cues to the user: info, warn, and error.

  • console.info(): similar to console.log(), some browsers may display a tooltip with the letter 'i' inside.
    console.info("Some helpful information");
  • console.warn(): displays a yellow warning icon. The text and background are also yellow.
    console.warn("A warning");
  • console.error(): displays a red cross icon. The text and background are also red.
    console.error("An error");

Depending on the severity of the logging statement, you can alternate between these methods.
