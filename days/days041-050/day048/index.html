JavaScript: Conditionals

The conditional statements in JavaScript work the same as in other languages. For example, the IF statement is declared as follows:

if (condition) {
  // Code if the condition evaluates to True
}
else {
  // Code if the condition evaluates to False
}

Suppose you want to check whether the user input is a specific number, you can use the following code:

const THE_ANSWER = 42;

let userGuess = document.getElementById('user-guess');
let text = document.querySelector("h4");

function theMeaning() {
 num = userGuess.value;
 if (num == THE_ANSWER) {
  text.textContent = `YOU'RE RIGHT! The answer to the life,
    the universe and everything else is 42!`;
 }
 else {
  text.textContent = `Your guess: ${num}, is not correct.
    Hint: try a number between 40 and 50`;
 }
}

(Don't forget to include the script tags)
