Bold and italics

There are several ways to make changes in the visual style of text. For beginners, this can be done in HTML.

The first option is using the bold <b></b> and italics <i></i> tags. This option only affects the visual style.

However, several people use screen readers for accessibility. For this reason, two other tags were introduced: strong <strong></strong> and emphasis <em></em> tags. These tags change the visual style of the text, but also its semantics, meaning that screen readers will change their tone of voice to pronounce words within these tags.
