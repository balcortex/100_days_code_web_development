To add an external link (any webpage on the internet), include the protocol (http, ftp, etc.) on the href attribute of the <a> tag.

For example: https://www.google.com. Clicking this link will open the Google website.

To add an internal link, include the path of the .html file relative to the current file.

For example: ./pages/about.html. In this example, the about.html is within the pages folder in the root directory.
