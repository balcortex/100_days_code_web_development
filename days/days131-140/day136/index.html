Day 136 - HTML: Input Elements - Pt. 2

Type attribute

There are different input elements, each one for a unique usage. You can set the input type using the type attribute.

Text input

This is the default value. Allows the user to introduce any text, including numbers and special characters.

Email input

This field specializes just in email addresses. The benefits of using the email type include input validation and displaying a different keyboard on mobile devices, including the @ symbol.

Password input

This input hides the entered information behind asterisks or bullet points (depending on the browser).

Number input

The number input only accepts numbers, ignoring any other characters the user tries to enter.

Date input

The date input allows the user to enter or pick a date.
