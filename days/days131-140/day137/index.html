Day 137 - HTML: Input Elements - Pt. 3

Radio type

Radio buttons allow us to create multiple options that the user can choose only one of.

When we select one of the radio buttons and then select another, it will deselect the first one. Radio buttons know to do this because they have the same name attribute



Checkbox type

Checkboxes are similar to radio buttons, allowing users to choose from a set of predefined options. But unlike radio buttons, they allow multiple options to be selected simultaneously.



Color type

The color input provides a user interface element that lets a user specify a color, either by using a visual color picker interface or by entering the color into a text field in hexadecimal format.

Range type

The range input lets the user specify a numeric value, which must be no less than a given value (min) and no more than another given value (max). The precise value, however, is not considered relevant. This is typically represented using a slider or dial control rather than a text entry box like the number input type.
