Day 134 - CSS: Variables - Pt. 3

Custom themes

In addition to allowing us to create global custom properties, the :root selector gives us a way to add themes to our pages.

For instance, let's add a light and a dark theme to our CSS document:

:root.light {
 --border-btn: gray;
 --color-base-bg: #ebeefd;
 --color-base-text: black;
 --color-btn-bg: white;
}

:root.dark {
 --border-btn: white;
 --color-base-bg: black;
 --color-base-text: white;
 --color-btn-bg: black;
}

We defined the same custom properties on both themes but with different values. When the light theme is active, the --color-base-text property will be set to black; when the dark theme is active, this same property will have the value of white.

The rest of the CSS document looks like this:

...
body {
 background-color: var(--color-base-bg);
 color: var(--color-base-text);
}
...

Now, we need to change the class of the :root element when a button is pressed. This is done in JavaScript:

const root = document.documentElement; // the :root element
document.querySelector('button').addEventListener('click', toggleTheme);

function toggleTheme(e) {
 const theme = root.className;
 root.className = theme === 'dark' ? 'light' : 'dark';
}
