Day 131 - CSS: Functions

Like any programming language function, CSS functions are reusable pieces of code that perform specific tasks. Similarly to other languages, functions in CSS are declared with parentheses, and the arguments are passed inside them.

For instance, you may have used the following syntax:

color: rgb(0, 42, 255);

Here, rgb() is a function that accepts numbers as arguments and calculates the corresponding rgb color.

Unlike programming functions, CSS functions cannot be created. Instead, the language comes with a list of pre-made functions. There are a lot, so here we will just cover a few.

calc()

This function allows us to perform basic math inside the CSS rules. You can use different units without any problem, for example, em and vh:

padding: calc(3px + 1em - 1vh);

min()

This function returns the minimum value from the given values:

width: min(200px, 4rem, 100%);

max()

This function works the same as min(), except that it returns the largest value:

height: max(150px, 2rem, 50%);
