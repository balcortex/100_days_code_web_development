Day 138 - HTML: Input Elements - Pt. 4

Other input elements

Other input elements do not share the <input> tag as the radio buttons or checkboxes do. Instead, they use their own tag, but they also provide support for user interaction.

Select

The <select> HTML element represents a control that provides a menu of options. It behaves mostly like the radio buttons because you can only select one option. The advantage of the select element is that it expands when the user clicks on it and collapses once the selection is made.

Text area

The <textarea> element is similar to the input text element. Still, it can span many lines instead of using a single row. Also, the user can make the element bigger by dragging the bottom right corner as he likes.

This element is mostly used to collect user comments or reviews.
