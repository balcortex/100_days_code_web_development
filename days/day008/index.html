Internal CSS Style

Inline style could be useful if you want to change a few attributes on specific elements.

However, if you want to apply the same style to several elements, internal CSS is a better choice.

With internal CSS, you can define a single style for all the elements at once.

The internal CSS is defined within the head element, with a similar syntax as the inline CSS.

In addition, you can still use the inline CSS to redefine the style of a specif element.
