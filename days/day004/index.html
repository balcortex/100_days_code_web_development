Lists

There are two types of lists:

  • Ordered lists
  • Unordered lists

Steps to create a list:

  1. Start with the opening tag:
    • <ol> for Ordered Lists.
    • <ul> for Unordered Lists.
  2. Use <li> an item </li> to add an item to the list.
  3. Repeat step 2 as necessary.
  4. End with the corresponding closing tag:
    • </ol> for Ordered Lists.
    • </ul> for Unordered Lists.
