Flexbox Align-Items (Column)

If the flex-direction is set to column, the justify-content property controls the position of the items on the y-axis. You can use the align-items property to move the items on the x-axis.

Next are some examples of the different values of the align-items property. The configuration is the same for all cases (except the last one, which it is explained there). Item 2 and Item 4 have different font sizes to modify their size.

align-items: stretch (default). Change the items' width to match the container.
