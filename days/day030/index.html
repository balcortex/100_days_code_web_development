Flexbox Align-Content

If the flex-direction is set to row, the justify-content property controls the position of the items on the x-axis. You can use the align-content property to move the items on the y-axis.

Next are some examples of the different values of the align-content property. The configuration is the same for all cases; the items are not allowed to grow (on the x-axis), and Item 2 and Item 4 have different font sizes to modify their height.

align-content: stretch (default). Change the items' height to match the container.
