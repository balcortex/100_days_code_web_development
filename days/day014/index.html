Block vs Inline Display

Every HTML element has a default display type, which is the element's positioning in the canvas (i.e., the web page's body). There are two types of display values:
  • Block
  • Inline

(In fact, there are three types; the other is a spin-off version of the inline display, which will be covered later).

Some HTML elements have a default display value of block: headings (h1), paragraphs (p), lists (ol), and images (img). Whereas others have the default of inline: anchors (a), strong (strong), emphasize (em).

However, the display value can be set with CSS. For example, we can make a strong element having its own line changing its display value to block.
