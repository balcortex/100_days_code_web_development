Day 104 - Project: Calculator, Pt. 2

This day, I worked on the style, arranging the buttons, changing the font size, adding borders, etc.

First, I introduced several divs inside the `buttons` div, one for each button row. The purpose of these divs is to act like a flex container, so all the buttons will stretch to match the calculator's size:

...
<div id="buttons">
 <div class="buttons-row">
  <button id="button-7">7</button>
  <button id="button-8">8</button>
  <button id="button-9">9</button>
  <button id="button-div">/</button>
 </div>
 ...

In the CSS document, we define the outer `buttons` div as a flex container with column direction, which will position each row of buttons one below the other. The gap property adds little space between the rows:

#buttons {
 display: flex;
 flex-direction: column;
 gap: 2px;
}

Next, in the `buttons-row` selector, we set the display as flex (the default direction is row), placing the buttons in the row one after the other, left to right. Also, we add some space between the buttons on the row with the gap property:

.buttons-row {
 display: flex;
 gap: 2px;
}

Finally, set the flex property to 1 on each button so it can fit the entire container (i.e., the calculator width):

button {
 flex: 1;
 font-size: 20px;
}
