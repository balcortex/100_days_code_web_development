Let's complete the JS code to add the support for all the operations.

First, we need a variable to store the operator selected by the user, which is declared at the start of the document:

let operator = '';

When the user presses an operator button, we store the selected operation:

...
case 'op':
 numMem = screen.innerText; // save the current number on screen to memory
 numCur = '0'; // clear the current number
 operator = value;
 break;
...

And when the user presses the equal button, we call a function that operates on both the entered numbers and the selected operator:

...
case 'eq':
 screen.innerText = performOperation(+numMem, +numCur, operator);
 numMem = screen.innerText;
 numCur = '0'; // clear the memory
 break;
...

The function `performOperation` is straightforward:

function performOperation(num1, num2, operator) {
 switch (operator) {
  case 'div':
   return num1 / num2;
  case 'mul':
   return num1 * num2;
  case 'dif':
   return num1 - num2;
  default:
   return num1 + num2;
 }
}
