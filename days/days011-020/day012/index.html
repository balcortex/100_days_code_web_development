CCS ID Selector

There is another selector in CSS, apart from the Type Selector and the Class Selector, which is the ID Selector.

The ID attribute is declared similarly to a Class attribute inside the opening tag with the syntax id="id_name" . However, an ID must be unique, which means two HTML elements cannot share the same ID.

In addition, the ID Selector has the highest specificity (i.e., importance). If an element has colliding CSS attributes, those declared in the ID selector will be applied. The specificity order is the following:

  1. ID
  2. Class
  3. Type

To define an ID Selector in CSS, use the # symbol before the id_name.
