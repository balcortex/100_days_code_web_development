Parts of the Box Model

The box model has four elements:

  • Margin: distance between boxes (has no color).
  • Border: the visual limits of the box (i.e., the frame - can have a color).
  • Padding: distance between the content and the border (can have a color).
  • Content: the actual content of the box.

For example, consider the following HTML elements:

10-pixel border, no margin (default value), no padding (default value)
30-pixel border, different margins, 20-pixel padding.
