The Box Model (This is a box)

Every HTML element in a web page is contained in a box. The outer blue box contains the html element, the parent element. Within this blue box is the body element, colored green. Following these two boxes, all the other elements make the web page, contained in red boxes.

  • By default, boxes expand to match the width of the parent.
  • And their height shrinks to their content.

Links and images are also contained in boxes: A link to Google is a box

.

Knowing how the elements are positioned to improve the page's visuals is essential.
