Day 59 - JavaScript: Arrays, Pt. 5

Other approaches exist for looping over the array's items without using the index to access them.

The first approach is to use the for ... of, which iterates over the values of an array (any iterable indeed). This method is preferred over index looping, as it is more clear, unless you also need the index to perform a calculation.

let colors = ['Red', 'Blue', 'Green'];

for (let color of colors){
 console.log(color); // Red, Blue, Green
}

Another similar statement, the for ... in, iterates over the enumerable string properties. For instance, using the for ... in the same code as before yields:

for (let color in colors){
 console.log(color); // 0, 1, 2
}

Of course, you can use these indices to access the items:

for (let index in colors){
 console.log(index + ' ' + colors[index]); // 0 Red, 1 Blue, 2 Green
}

You can also use the for ... in to obtain the indices if they are not numbers. In this case, you cannot use a simple for loop, nor a for ... of loop, because they require numerical indices:

colors['noColor'] = 'Black';

for (let i = 0; i < colors.length; i++){
 console.log(i + ' ' + colors[i]); // 0 Red, 1 Blue, 2 Green
 // Black is not listed
}

for (let color of colors){
console.log(color); // Red, Blue, Green
 // Black is not listed
}

for (let index in colors){
 console.log(index + ' ' + colors[index]); // 0 Red, 1 Blue, 2 Green, noColor Black
}
