Day 58 - JavaScript: Arrays, Pt. 4 & The FOR Loop

First, let's define what the FOR loop does. Imagine you want to double a variable 10 times and perform other operations every time the value changes. One approach is to repeat each fragment of the code 10 times:

let i = 2;
console.log(i); // 2
i = i * 2;
console.log(i); // 4
...

Of course, this approach is not sustainable. Imagine if you have to do the same thousands of times! Fortunately, the FOR loop simplifies this task:

let i = 2;
for (let k = 0; k < 10; k++){
 i *= 2;
 console.log(i); // 4, 8, 16, 32...
}

Inside the declaration of the FOR loop, the first statement is executed one time before at the start of the loop; the second statement controls whether the loop stops (when the condition evaluates to false, the loop stops), and finally, the last statement is executed at the end of each iteration (generally controls the looping variable).

The FOR loop can be used to loop over the elements of an array:

let colors = ['Red', 'Blue', 'Green'];
for (let k = 0; k < colors.length; k++){
 console.log(k + ' ' + colors[k]);
 // 0 Red
 // 1 Blue
 // 2 Green
}

Note: This is a very simple technique for looping over the elements of an array; there are other more advanced approaches.
