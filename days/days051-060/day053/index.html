JavaScript: The + Operator, Pt. 2

The plus operator can also be used to implicit cast types.

Number to string

If the operands are a string and a number, the number is converted to a string. Then, the plus operator performs a concatenation.

let x = "Hello" + 42; // 42 is converted to a string
console.log(x); // Hello42
console.log(typeof x); // string

String to number

To convert a string to a number, use the plus operator before the string (or variable).

let y = 42 + "3"; // 42 is converted to a string, as before
console.log(y); // 423
console.log(typeof y); // string

let z = 42 + +"3"; // +"3" is equivalent to Number("3")
console.log(z); // 45
console.log(typeof z); // number

It's common to use the plus operator for implicit casting, so beware of it.
