JavaScript: The + Operator

The plus operator is overloaded, which means it behaves differently depending on the data type of the operands.

Numbers

For numbers, the operation is straightforward: perform the sum of both numbers:

let x = 3;
let y = 5;
let z = x + y;
console.log(z); // 8

Strings

For strings, the plus operator performs a concatenation (i.e., join the strings together):

let str1 = "Hello";
let str2 = "World!";
let str3 = str1 + str2;
console.log(str3); // HelloWorld!

Note that the result "HelloWorld!" does not include a space between the words.
