Day 56 - JavaScript: Arrays, Pt. 2

An array can also be defined as empty, and then add the elements using the index:

let myArray = []; // []
console.log(myArray.length); // 0
myArray[0] = 'zero'; // zero
console.log(myArray.length); // 1
console.log(myArray[0]); // zero

However, this could create 'holes' in the array if you use an index greater than the length:

myArray[2] = 'two'; // two
console.log(myArray.length); // 3
console.log(myArray[2]); // two
console.log(myArray[1]); // undefined

The preferred way to add an element to (the end of) an array is using the method push:

myArray.push('three'); // 4
console.log(myArray.length); // 4
console.log(myArray[myArray.length - 1]); // three

Also, removing the last element of an array is straightforward; just use the method pop:

myArray.pop(); // three
console.log(myArray.length); // 3
console.log(myArray[myArray.length - 1]); // two
