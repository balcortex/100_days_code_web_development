JavaScript: Data Types

There are eight data types in JavaScript (seven are primitive):

  1. Number
  2. String
  3. Boolean
  4. BigInt
  5. Undefined
  6. Null
  7. Symbol
  8. Object (non-primitive)

The most common ones, at least for beginner developers, are number, string, and boolean.

To check the type of any variable, you can use the operator (not a function) typeof.

Let's use the typeof operator to find the type of the following variables:

let n = 42;
let text = "Hello World!";
let isTurnedOn = true;

Note: typeof and typeof() are similar. The parentheses are used to group operations (like in math operations) but are not required.
