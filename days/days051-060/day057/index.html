Day 57 - JavaScript: Arrays, Pt. 3

There are other useful methods of Array objects. One of them is the toString() method, which joins all the elements together:

let fruits = ['Bananas', 'Grapes', 'Strawberries'];
console.log(fruits.toString()); // 'Bananas,Grapes,Strawberries'

You can use this method to populate HTML elements. For example, the following paragraph shows my Top 3 favorite fruits:

let fruitsField = document.getElementById('fruits');
fruitsField.textContent = fruits.toString();

If you need some simple pre-processing of the array, use the join() method, which puts the specified string between each item:

let fruitsField2 = document.getElementById('fruits-2');
fruitsField2.textContent = fruits.join(', ');

That looks better, but it still needs improvement. We need to insert the word `and` between the last one and the one before. To achieve this, we can use the slice() method to select the first two fruits but not the last. Then, the join() method, and finally, simple string concatenation with the last element of the fruits array:

let fruitsField3 = document.getElementById('fruits-3');
fruitsField3.textContent = fruits.slice(0,2).join(', ') + ', and ' + fruits[fruits.length - 1];

Note: this is only one approach to achieve this. Several other approaches (even more optimal) can be used to achieve the same result.
