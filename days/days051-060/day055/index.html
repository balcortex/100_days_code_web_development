Day 55 - JavaScript: Arrays, Pt. 1

An array is a special type of variable that can hold more than one value.

Imagine that you want to store the name of your pets in the code; instead of defining several, values you can use an array to store all the names in a single variable:

let pets = ["Max", "Daisy", "Lucy", "Molly", "Bella"];

The type of an array variable is object:

console.log(typeof pets); // object

You can access any individual element by its index, starting at zero. For example, to access the second name, use the index 1:

console.log(pets[1]); // Daisy

To retrieve the number of items in the array, use the length property:

console.log(pets.length); // 5

This property can be useful if you want to access the last element in the array and don't know the length beforehand:

console.log(pets[pets.length - 1]); // Bella
