Day 60 - JavaScript: Arrays, Pt. 6

Another more advanced and handy approach to looping over the elements of an array is using the forEach() method of the Array object.

This method receives a callback function as a parameter and then executes the function for each item:

let colors = ['Red', 'Blue', 'Green'];

function myLog(element){
 console.log(element);
}

colors.forEach(myLog); // Red, Blue, Green

You can also include the index in the callback function:

function myLog2(element, index){
 console.log(index + ' ' + element);
}

colors.forEach(myLog2); // 0 Red, 1 Blue, 2 Green

And you can even include a reference to the array itself:

function myLog3(element, index, array){
 console.log(`Index ${index} of ${array.length - 1} has the value ${element}`);
}

colors.forEach(myLog3);
// Index 0 of 2 has the value Red
// Index 1 of 2 has the value Blue
// Index 2 of 2 has the value Green

There are several ways to loop over the elements in an array. Choose the best suited for your needs.
