Number range validation

Like character length validation, we often want to control the range of values users can enter into number-based inputs.

We can set a lower bound with the min attribute, an upper bound with the max attribute, or a range combining both attributes simultaneously. Note that these attributes only work with number-based input form controls such as number, time, and dates.

For instance, suppose that you are working in a form for employees. You know that employees can't be under 18 or over 100 years old (in the best-case scenario), so you can use both the min and max attributes to allow only values between 18 and 100.

<form action="#" method="post">
 <label for="age">Age:</label>
 <input type="number" id="age" name="age" min="18" max="100">
 <input type="submit">
</form>
