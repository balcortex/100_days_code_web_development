Day 150 - CSS: Grid - Pt. 3

Rows

Unlike columns, rows are created automatically to fit all the items once the number of columns has been specified. However, we can still decide each row's size (height).

To specify the height of the rows, use the grid-template-rows property, followed by the heights of the rows. For instance, to give two rows the height of 50px and 100px, respectively, use the following code:

.container {
 display: grid;
 grid-template-columns: 50px 80px;
 grid-template-rows: 50px 100px;
}

The HTML code is as follows:

<div class="container">
 <div>Item 1</div>
 ...
 <div>Item 4</div>
</div>

The container has four children, and we have defined two columns so the layout will be a 2x2 grid. The rows are 50px and 100px high:
