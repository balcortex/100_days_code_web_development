Day 149 - CSS: Grid - Pt. 2

Columns

To specify the number of columns (and their width), use the grid-template-columns property, followed by the widths of the columns. For instance, to create two columns, one of 50px and the other of 80px wide, use the following CSS code:

.container {
 display: grid;
 grid-template-columns: 50px 80px;
}

The HTML document is like this:

<div class="container">
 <div">Item 1</div>
 ...
 <div">Item 5</div>
</div>

Note that, depending on the total number of children, the rows are automatically generated. In this case, there are five items, so the layout is a 3x2 grid (with a blank space):

Item 1
Item 2
Item 3
Item 4
Item 5
