Minimum text length validation

Sometimes, we want the users to enter a minimum amount of text for passwords, for example.

The minimum length validation will prevent the user from inputting less than the desired characters. For example, in a password field, we want the user to enter eight characters at least:

<form action="#" method="post">
 <label for="password">*Password:</label>
 <input type="password" id="password" name="password" minlength="8" required>
 <button type="submit">Submit</button>
</form>
