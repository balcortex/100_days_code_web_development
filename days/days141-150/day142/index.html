Form validation

Validations allow us to set specific constraints determining what data users can enter into an input. The form will only be sent if the entered information is valid.

Required validation

We often want to ensure that specific fields are correctly filled before submitting the form, for instance, the email and password in a login form. To make a field required, we add the required attribute:

<form action="#" method="post">
 <label for="email">*Email:</label>
 <input type="text" id="email" required>
 <button type="submit">Submit</button>
</form>

We should always indicate which fields are required. This will often be done by adding an asterisk (*) to the required field label.
