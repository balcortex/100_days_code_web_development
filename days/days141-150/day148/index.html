Day 148 - CSS: Grid - Pt. 1

Grid, like Flexbox, is a layout tool. The difference is that Grid was primarily designed for the easy placement of items in a two-dimensional layout (but it can be used for one-dimensional layouts, too).

You will notice a lot of similarities between Flex and Grid, but you will also come across many differences between the two and opinions on how each module should be used. For example, if you've been struggling to get your Flex items to match evenly in size, Grid can make this type of layout much more effortless.

Grid container

We can think about CSS Grid in terms of a container and items. When you make an element a grid container, it will “contain” the whole grid. In CSS, an element is turned into a grid container with the property display: grid.

.container {
 display: grid;
}

In this example, the parent element becomes a grid container, and each of the direct child elements below it automatically become grid items:

<div class="container">
 <div>Item 1</div>
 <div>Item 2</div>
 <div>Item 3</div>
 <div>Item 4</div>
</div>
