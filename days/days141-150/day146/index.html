Pattern validation

To ensure the user enters the correct information, sometimes we want to ensure the data matches a specific pattern, for example, a credit card number or a postal code.

To add a pattern validation, add the pattern attribute with the regular expression (regex) that defines the expected data shape. Regex is a complex topic, but you can always search the web for common patterns.

For instance, if you want to validate a 5-digit postal code, use the following regular expression:

<form action="#" method="post">
 <label for="zip-code">Zip Code:</label>
 <input type="text" id="zip-code" name="zip-code" pattern="(\d{5})"
 placeholder="65251" title="Please enter a 5-digit zip code">
 <input type="submit">
</form>
