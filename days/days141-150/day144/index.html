Maximum length validation

The maximum length validation works differently than the minimum length validation. Instead of informing the user, the browser will prevent users from entering more characters than the declared maxlength attribute value.

For instance, to limit a field to no more than six characters, we set the maxlength to 6:

<form action="#" method="post">
 <label for="account-no">*Account No.:</label>
 <input type="text" id="account-no" maxlength="6" required>
 <button type="submit">Submit</button>
</form>

The browser will prevent the user from entering more than six characters, omitting the rest once the length is reached. If the user enters the following sequence "123456789", the browser will accept only the sequence "123456", discarding the other characters.

Note: you can use both the min and the max validation simultaneously.
