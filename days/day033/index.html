Flexbox Wrap

Until now, we've seen that flex containers set elements in a single dimension (a row or a column). However, we can allow flex items to wrap onto multiple lines (by adjusting the items' width or height, depending on the direction). This behavior is controlled by the value of the property flex-wrap, whose default value is nowrap, which, you guessed right, doesn't allow wrapping:
