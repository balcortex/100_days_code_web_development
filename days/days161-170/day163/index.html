auto-fit and auto-fill

Imagine that you want to give your grid a flexible number of columns based on its size. For example, if the grid is only 150px wide, you may only need one column. If it's 300px, you may want two columns, and so on.

You can easily accomplish this task using the auto-fit (or auto-fill) value, which is a part of the repeat() function:

.container {
 display: grid;
 grid-template-rows: repeat(2, 1fr);
 grid-template-columns: repeat(auto-fit, 150px);
}
