Day 167 - JavaScript: Prototype - Pt. 3

The ultimate prototype

Until now, we have seen that all objects created with the same constructor share one prototype. And sharing the prototype means sharing all its methods and properties. But can a prototype also have a prototype? Let's check the following code:

function User(name, isAdmin) {
 this.name = name;
 this.isAdmin = isAdmin;
}

let user1 = new User("John", false);

console.log(user1.valueOf()); // User {name: 'John', isAdmin: false}

Where did the method valueOf() come from? It results that all objects have as their last prototype the Object.prototype:

console.log(Object.getPrototypeOf(User.prototype) === Object.prototype); // true

Thus, all objects have access to the methods and properties of the Object.prototype, including the valueOf() method that we saw earlier.
