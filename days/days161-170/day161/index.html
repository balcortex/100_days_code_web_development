Minimum and maximum track sizes: minmax()

On the last day, we saw that we could use min() to set the maximum size and max() to set the minimum size. But what if we want to set both attributes at a time? Enter minmax()

The function minmax() is exclusive of the Grid layout, and is a straightforward manner to set the minimum and maximum sizes a grid track can be. Unlike the min() and max() functions, we can use static values for both arguments.

Take a look at the following code:

.container {
 display: grid;
 grid-template-rows: repeat(2, 1fr);
 grid-template-columns: 50px 100px minmax(150px, 300px);
}

With our grid-template-columns set with minmax() values, the third column item's width will grow and shrink with the grid container as it resizes horizontally. However, as the grid shrinks, the column tracks will stop shrinking at 150px:
