Minimum and maximum track sizes: clamp()

Unlike minmax(), clamp() is a CSS function that can be used anywhere, not just within grid containers. This function adds another parameter to the minmax(), allowing our item to resize itself until it reaches one of the minimum or maximum threshold values.

The syntax is the following:

clamp(minimum-size, ideal-size, maximum-size)

Since clamp() purpose is to create a flexibly sized track with constraints, we want to use a dynamic value for the "ideal size" argument and typically a static size for the minimum and maximum size. However, it is possible to use a dynamic value here too.

Let's take a look at the following code:

.container {
 display: grid;
 grid-template-rows: repeat(2, 1fr);
 grid-template-columns: repeat(3, clamp(50px, 30%, 150px));
}

Here, we specify that the three columns will have a width of 30% of their container unless this number is smaller than 50px or bigger than 150px. The container below has a width of 350px, which gives each column a width of 105px (30%):
