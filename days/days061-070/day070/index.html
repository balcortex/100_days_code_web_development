Day 70 - JavaScript: DOM Manipulation, Pt. 6

Element creation

JavaScript also allows us to use a script to place new elements on the DOM. It is important to note that the actual HTML is not modified; just the elements are added to the DOM when the script is executed.

There are two steps to add new elements to the DOM. First, you need to create them using the document.createElement() method. In this phase, you can style the element before adding it to the DOM. Second, use the parentNode.appendChild() to add the element into the parent element.

Suppose you have the following HTML document:

<h1>The title</h1>
<div id="container">
 <div class="content">This is the first paragraph.
 This content and the title exist in the HTML document.</div>
</div>

That should render like this:

The title

This is the first paragraph. This content and the title exist in the HTML document.

You can add another paragraph using the following code:

const container = document.querySelector('#container'); // The parent node

const p = document.createElement('p'); // Create the new element in memory
p.textContent = "This is the second paragraph."; // Add the content
p.style.color = 'blue'; // Style the element
container.appendChild(p); // Add the new element to the parent node

The result should be rendered as:

The title

This is the first paragraph. This content and the title exist in the HTML document.

This is the second paragraph.

You can create any element, even nested elements. But remember that this content is not added to the HTML document, just to the DOM.
