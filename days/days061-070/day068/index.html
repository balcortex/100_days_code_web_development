Day 68 - JavaScript: DOM Manipulation, Pt. 4

We should be aware of some caveats when using the document.querySelector() method. For example, suppose you have the following HTML document, which is composed of three paragraphs:

<p id="p1">This the 1st paragraph.</p>
<p id="p2">This the 2nd paragraph.</p>
<p id="p3">This the 3rd paragraph.</p>

This HTML document should be rendered like this:

This the 1st paragraph.

This the 2nd paragraph.

This the 3rd paragraph.

Suppose you want to add inline style to all the paragraphs. The following approach will not work because the querySelector() returns the first match only:

const p = document.querySelector('p');
p.style.color = 'blue';

The result would be:

This the 1st paragraph.

This the 2nd paragraph.

This the 3rd paragraph.

As all the paragraphs have an ID, you could use this selector to target each one. But if you had 100 paragraphs, something else would be needed.

const p2 = document.querySelector('#p2');
p2.style.color = 'red';

The result of running the two previous code fragments would be:

This the 1st paragraph.

This the 2nd paragraph.

This the 3rd paragraph.

Finally, the best option is to use the document.querySelectorAll(), which returns an array of elements. Then, you can use a for loop to iterate over the elements:

const paras = document.querySelectorAll('p');
for (para of paras){
 para.style.color = 'green';
}

The final result should be:

This the 1st paragraph.

This the 2nd paragraph.

This the 3rd paragraph.
