Day 67 - JavaScript: DOM Manipulation, Pt. 3

In addition to changing the content of the elements, you can also add inline style using JavaScript.

Suppose you have the following HTML document:

<h1>Title</h1>
<p>Some text</p>

That should render as:

Title

Some text

To add inline style, you must target the element and specify the property. For example, to center the title, convert it to uppercase, and change its color to red, use the following code:

const title = document.querySelector('h1');
title.style.textAlign = 'center';
title.style.textTransform = 'uppercase';
title.style.color = 'red';

Similarly, to change the font size of the paragraph and its font style to italics, use the following code:

const par = document.querySelector('p');
para.style.fontSize = '12px';
para.style.fontStyle = 'italic';

The result will be rendered as:

Title

Some text

The new HTML document would be:

<h1 style="text-align: center; text-transform: uppercase; color: red;">Title</h1>
<p style="font-size: 12px; font-style: italic;">Some text</p>

JavaScript can add inline style, but this approach is seldom used because it can get messy. There are other approaches to apply style that will be covered next.
