Day 66 - JavaScript: DOM Manipulation, Pt. 2

Again, suppose you have the following HTML document:

<div id="container">Container
 <div class="header">Header</div>
 <div class="main">Main</div>
 <div class="footer">Footer</div>
</div>

A web browser would render the previous HTML code as:

Container
Header
Main
Footer

Now, let's change the content of the `main` div using JavaScript. First, we need to target the element. There are several ways to do this; let's review some.

1. Target the `container`, obtain its children, and select the index that corresponds to the `main` div:

console.log(document.querySelector("#container").children[1]); // div.main

2. Target the `container`, select the first child using the firstElementChild property, and move to the next sibling using the nextElementSibling property:

console.log(document.querySelector("#container").firstElementChild.nextElementSibling);
// div.main

3. Target the `container`, select the last child using the lastElementChild property, and move to the previous sibling using the previousElementSibling property:

console.log(document.querySelector("#container").lastElementChild.previousElementSibling);
// div.main

4. Target the `main` class directly, which returns the first element it finds. This approach is the most straightforward, so let's choose this method and save it to a variable to use next:

const mainField = document.querySelector(".main");
console.log(mainField); // div.main

Once you have targeted the element, use the textContent property to store the new text that the element should display:

mainField.textContent = 'This is the new content for the main section';

Now, the web browser should render the web page as:

Container
Header
This is the new content for the main section
Footer

And that is all for modifying the content of elements on the DOM.
