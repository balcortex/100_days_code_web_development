Day 69 - JavaScript: DOM Manipulation, Pt. 5

Apart from adding and removing inline style, there is another approach to modifying the elements' style using JavaScript: adding and removing classes. This approach is cleaner because all the styling is separated from the HTML document, and if you have to apply several lines of style to many elements using inline style, the code can quickly become messy.

Suppose you have the following HTML document (and another CSS document that I'll omit here because of limited space):

<p id="p1">This is the 1st paragraph.</p>
<p class="big-red" id="p2">This is the 2nd paragraph.</p>
<p id="p3">This is the 3rd paragraph.</p>

That should render like this:

This is the 1st paragraph.

This is the 2nd paragraph.

This is the 3rd paragraph.

You can use the classList.add() and classList.remove() methods on the targeted elements to add and remove the style class, respectively:

const p1 = document.querySelector("#p1");
p1.classList.add('fancy'); // Add the `fancy` class to the 1st paragraph.

const p2 = document.querySelector('#p2');
p2.classList.remove('big-red'); // Remove the `big-red` class from the 2nd paragraph.

Also, there is the classList.toggle() method that will add the class (i.e., activate the style) to the element if the class is not active and will remove (i.e., deactivate the style) if the class is inactive:

const p3 = document.querySelector('#p3');
p3.classList.toggle('high-contrast'); // Add the `high-contrast` class to the 3rd paragraph (because it is not active at the moment).

The final result should look like this:

This is the 1st paragraph.

This is the 2nd paragraph.

This is the 3rd paragraph.

The DOM will look much cleaner because it will only contain the classes and not the full inline style in each element:

<p class="fancy" id="p1">This is the 1st paragraph.</p>
<p class="" id="p2">This is the 2nd paragraph.</p>
<p class="high-contrast" id="p3">This is the 3rd paragraph.</p>
