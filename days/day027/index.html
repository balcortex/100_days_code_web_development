Flexbox Direction

Until now, we have worked only with the properties of the flex items, but the containers have their properties too. The flex-direction is the most important property for a flex container (apart from display: flex).

The default value for flex-direction is row, where the items are set one right to the other. By using the row value (and allowing the items to grow), the result is the following:
