Flexbox

There are many ways to move elements around on a web page.

Flexbox (flexible box layout mode) is a relatively new way of manipulating elements in CSS, and its debut was revolutionary. Many modern web pages use flexbox to re-arrange elements, mostly on navigation links (headers and footers).

Flexbox arranges elements into a single row by default. The elements within this row will flex (i.e., grow or shrink) based on the rules you define.

For example, suppose we have three div elements (with the default block display property). Without using flexbox, the items will be shown like this:

By using the flexbox layout, the result is the following:

(1) Flex-item

To use the flexbox layout mode, use the display property with the flex value.

In flexbox, the element that uses the display: flex property is called the flex-container (the pink rectangle), and the elements within the flex-container (blue rectangles) are called the flex-items.
