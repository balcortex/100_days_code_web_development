Day 154 - CSS: Grid - Pt. 7

Grid lines

Whenever we create grid tracks, grid lines are created implicitly. These grid lines are used to position grid items.

Every grid has start and end lines. The lines are numbered from left to right and top to bottom, starting at 1. So, in a 2x2 grid, the columns range from 1 to 3, as well as the horizontal lines.

Positioning - Multiple Columns

We can set the column where it starts and ends using the grid-column-start and grid-column-end properties on the actual item.

Suppose you have a 2x2 grid with three items and want the first item to span two columns. To achieve this, you must set the start of the item to 1 and the end to 3 (the line number 2 is the middle):

#item1 {
 grid-column-start: 1;
 grid-column-end: 3;
 }
