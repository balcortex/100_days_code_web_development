Positioning - Grid area

There is another approach to positioning the items on the grid. Instead of using grid lines to delimit the items, we can create a visual layout of the grid using words. That is, we make a blueprint of the final layout.

First, we need to define the areas in the container using the grid-template-areas property. This property's value will be a series of strings (one for each row) in which we map each cell (each column of the row) to a specific area name.

For instance, we can obtain the same layout we used in the previous example using grid-template-areas with the following code:

.container {
 /* Other CSS code . . . */
  grid-template-areas:
  "area-1 area-1 area-1"
  "area-2 area-3 area-3"
  "area-2 area-3 area-3";
 }

Then, using these areas in each item (only the first item is shown; the other items are similar):

#item1 {
 grid-area: area-1;
}
