Positioning - Span keyword

Instead of defining the start and end of an item in the grid, we can use the span keyword to define the start and the number of cells the item will fit in.

Also, you can even define the endpoint and use the span keyword to specify where the item is supposed to start.

Suppose you have a 3x3 grid with three elements to exemplify this. You want the first item to fit the entire first row (3 cells), the second to fit the remaining space of the first column (2 cells), and the third item to make up the rest of the grid (4 cells).

The CSS code is as follows:

#item1 {
 grid-column-end: span 3;
}
#item2 {
 grid-row-end: span 2;
}
#item3 {
 grid-column-end: span 2;
 grid-row-end: span 2;
}

The benefit of using this syntax is that the number of cells (the area) composing each item is immediately apparent.
