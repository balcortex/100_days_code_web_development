Day 152 - CSS: Grid - Pt. 5

Implicit grid - columns

With default settings, the items are arranged into the grid, filling the columns and then filling the rows until all items are included. This is the default implicit grid behavior.

We can change the grid behavior to first fill the rows and then the columns, using the grid-auto-flow property, setting its value to column (row is the default):

.container {
 grid-template-columns: 50px 80px;
 grid-template-rows: 50px 100px;
 grid-auto-flow: column;
 grid-auto-columns: 200px;
 }

Every new item for which we haven't defined an explicit layout will be placed in the next column (that is, to the right). The columns that haven't had their width defined with the grid-template-columns property will follow the width of the grid-auto-columns property.

As the CSS defines, we have an explicit grid layout of 2x2, but the actual container declared in the HTML has five items:

<div class="container">
 <div>Item 1</div>
 ...
 <div>Item 5</div>
 </div>

The new item will be added to a new column whose width is defined by the grid-auto-columns property.
