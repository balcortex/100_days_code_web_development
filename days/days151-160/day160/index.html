Minimum and maximum track sizes: min() and max()

When working with dynamic sizes, we often don't want the items to stretch indefinitely or shrink too much, so we can use the min() and max() functions to decide how small and large the item should be.

For instance, look at the following code:

.container {
 display: grid;
 grid-template-rows: repeat(2, min(40px, 50%));
 grid-template-columns: 50px max(50px, 30%) max(50px, 40%);
}

Starting with the rows, we define two rows that will be either 40px high or 50% of the container's height, whichever number is the smallest. That is, we define 40px as the maximum height unless the container is smaller than 80px high.

Conversely, the width of the columns will be at least 50px; that is, we define the minimum width using the max() function.
