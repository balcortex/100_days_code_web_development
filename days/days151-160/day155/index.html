Day 155 - CSS: Grid - Pt. 8

Positioning - Multiple Rows

As with columns, we can define the start and end of an item regarding rows. The corresponding properties are grid-row-start and grid-row-end.

Remember that these properties belong to the item, not the container.

Following the previous example, we have a 2x2 grid with three items. Now we want the second item to span two rows, so all the available space is used in this grid.

Because Item 2 is the only item we position explicitly on the grid, we must define the row and column position. We want Item 2 right to Item 1, so we set the column start to 2 and the end to 3 (this is done in one line using the grid-column shorthand). Finally, we set the rows properties to 1 and 3, making Item 2 span two rows:

#item2 {
 grid-column: 2 / 3;
 grid-row-start: 1;
 grid-row-end: 3;
 }

As expected, Item 2 is next to Item 1 and has a height of two cells:
