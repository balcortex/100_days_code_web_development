Gap

Using the gap property, we can easily change the space between two adjacent rows (or columns). Gap sizes can be adjusted separately for rows and columns using the column-gap and row-gap properties.

For instance, to space all the rows and columns symmetrically by 30px, set the gap property directly to this number:

.container {
 display: grid;
 grid-template-columns: 50px 80px;
 grid-template-rows: 50px 100px;
 gap: 30px;
}

All the rows and columns will be 30px away from each other (without accounting for the distance to the container's border):
