The repeat function

Until now, we have seen that to make a 2x5 grid, we manually have to declare each row and column size individually. It's not a problem for small grids, but imagine that your grid has tens or hundreds of columns and rows; that would take some effort.

Luckily, CSS provides us with the repeat function, which allows us to define multiple rows or columns and their size without manually typing each size.

For example, imagine that your layout is a 10x30 grid, and you have to make each row and column the size of 15px. Using the standard syntax, you have to type 30 times the size for the columns, and 10 times the size for the rows, but using the repeat function, you only have to write two numbers for each property:

.container {
 display: grid;
 grid-template-columns: repeat(30, 15px);
 grid-template-rows: repeat(10, 15px);
 }
