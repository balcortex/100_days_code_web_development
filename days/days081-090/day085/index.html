Day 85 - Project: Rock, Paper, Scissors - Pt. 5

All the logic is (almost) done. Now, it's time to enhance the visuals.

The first thing I thought of doing was to replace the text buttons with image buttons, which look better. For this purpose, I browsed for some descriptive images (the source is in the code) and downloaded them to a folder named `image` in the root directory.

There are several ways to make a clickable image, and I chose the approach of putting the image inside the button itself to avoid changing the logic code:

<div id="buttons">
 <button id="rock"><img src="img/rock.png" alt=""></button>
 <button id="paper"><img src="img/paper.png" alt=""></button>
 <button id="scissors"><img src="img/scissors.png" alt=""></button>
</div>

Also, I added some CSS code to the buttons to shrink the images and make them look more appealing:

img{ height: 180px; }
button{ background-color: white; border-radius: 10px; margin-right: 10px; }
button:hover{ background-color: #e6e6e6; }
button:active{ background-color: #c0c0c0; }
button:disabled{ opacity: 0.5; background-color: #c0c0c0; }
