Day 090 - Project: Shopping list - Pt. 2

Several features can enhance the user experience while using the web page.

1. Clear the typing box when an item is correctly added to the list, which is done by setting the value to an empty string:

newItemField.value = '';

2. When an item is added, return the focus to the typing box so the user can start writing again without dragging the mouse to make a click in the box. Also, this can be done when the page is loaded the first time. To achieve this, use the element.focus() method:

newItemField.focus();

3. The user can also want to use the `enter` key on the keyboard to add the item instead of pressing the button on the screen. This task can be completed in several ways. I chose to add an event listener to the typing box that checks whether the pressed key is `enter`, and when it is, perform a click on the `add button` programmatically:

newItemField.addEventListener('keypress', e => {
 if (e.key == 'Enter') addBtn.click();
});

4. Perform input validation, as we don't want to add empty items to the list:

if (newItemField.value == '') return; // do not add empty items

Now, the user should feel more comfortable using the webpage.
