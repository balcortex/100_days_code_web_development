Day 83 - Project: Rock, Paper, Scissors - Pt. 3

We have the player's and the computer's choices whenever a button is pressed. Now, it's time to make the game logic, which will be handled by the playRound() function.

To simplify things, we define a variable that indicates that the player won, and we will change it only in the contrary case.

let playerWins = true;

Also, we need to check if there's a draw so we can exit the function right away:

if (playerChoice == computerChoice) {
 roundResultField.textContent = 'Round result: draw';
 return;
}

If there's no draw, we check the possible combinations that make the player lose the round and update the variable accordingly:

if ( playerChoice == 'rock' & computerChoice == 'paper' |
   playerChoice == 'paper' & computerChoice == 'scissors' |
   playerChoice == 'scissors' & computerChoice == 'rock') {
    playerWins = false;
}

And finally, the code updates a field on the webpage that displays who won the round:

if (playerWins) roundResultField.textContent = 'Round result: you win';
else roundResultField.textContent = 'Round result: you lose';
