Day 82 - Project: Rock, Paper, Scissors - Pt. 2

On day 81, I made the basic UI respond correctly to the user input; the next step is to make the computer choose its option. The computer has to randomly select between the three possibilities whenever the user clicks a button.

The approach I chose makes use of an array of available options:

const AVAILABLE_CHOICES = ['rock', 'paper', 'scissors'];

Then, using the Math module, we generate a random number and multiply it by three since the random() method produces a number between 0 and 1. Finally, we round down the number to the nearest integer, thus having the numbers 0, 1, and 2. We use this number as the index of the choices array.

function getComputerChoice() {
 return AVAILABLE_CHOICES[Math.floor(Math.random() * 3)];
}

Every time a button is clicked, a function named `playRound` is called, which gets both the player and the computer choices and updates the UI:

function playRound() {
 let playerChoice = this.id;
 let computerChoice = getComputerChoice();

 playerChoiceField.textContent = `You chose: ${playerChoice}`;
 computerChoiceField.textContent = `Computer chose: ${computerChoice}`;
}
