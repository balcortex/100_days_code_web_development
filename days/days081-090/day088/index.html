Day 87 - Project: Rock, Paper, Scissors - Pt. 7

Now, it's time for wiring the UI and the JavaScript code. I first added a blank image with the text `Choose Your Weapon` as a placeholder for the actual image, which will be replaced when the selection is made.

To allow easy access to both image fields, I added an ID for them: ' img-player` and `img-computer`.

<img id="img-player" src="img/blank.png" alt="">
...
<img id="img-computer" src="img/blank.png" alt="">

In the JavaScript file, first query both selectors (for now on, I will include one to save screen space):

const imgPlayerField = document.querySelector('#img-player');

Within the playRound() function, we set the attribute `src` to match the correct file, the same as the button ID. Also, we set the opacity to 1.0 (more on this later):

imgPlayerField.setAttribute('src', `img/${playerChoice}.png`);
imgPlayerField.style.opacity = 1.0;

To make the winner stand up, we set the opacity of the loser (or both if there's a draw) to 0.4:

imgComputerField.style.opacity = 0.4;
playerScore++;

And finally, update the score in the middle of the board (note the use of inner html required to render the text correctly):

gameScoreField.innerHTML = `Score<br>${playerScore} &dash; ${computerScore}`;

Also, we can add an element at the end of the body to indicate the winner. The code is straightforward and no longer fits here so that I will omit it.
