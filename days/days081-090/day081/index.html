Day 81 - Project: Rock, Paper, Scissors

In the next few days, I will be working on developing a web page that lets you play the `Rock, Paper, Scissors` game against the computer.

I've started with a basic user interface (UI) composed of three buttons and a text field. The buttons let you choose between the three possible options. The text field is used to output the selected option (I could have used the console to log the user choice, but I want to make a video to show the functioning, and displaying the output on the webpage itself is more convenient for this purpose).

Let's start with the HTML document, which is straightforward. The buttons are conveniently grouped within a div, which would allow the use of a for loop to attach the event listener. The other element is the text field:

<div id="buttons">
 <button id="rock">Rock</button>
 <button id="paper">Paper</button>
 <button id="scissors">Scissors</button>
</div>
<p id="player-choice">You chose: </p>

In the JavaScript file, first, we query all the buttons, which are the children of the div whose id=`buttons`. Next, we loop over the buttons array (an HTML collection, to be precise) to attach the event listener to each button. The listener is a simple function that, for now, updates the text field to output the choice corresponding to the button pressed:

const buttons = document.querySelector('#buttons').children;
// all the children (buttons) of the div with id='buttons'
const playerChoiceField = document.querySelector('#player-choice');

for (btn of buttons) {
 btn.addEventListener('click', playerSelection);
}

function playerSelection() {
 playerChoiceField.textContent = `You chose: ${this.id}`;
}
