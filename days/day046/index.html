JavaScript: Hello World! Pt. 3

The third way to make a "Hello World!" program in JavaScript is using the alert() method (the window.alert() method does the same).

The alert() method displays an alert box with a message and an OK button. It is used when you want to show information to the user and don't expect an answer.

The following code will show the text "Hello World!" on an alert box as soon as the page is loaded (don't forget to include the script tag):

alert('Hello World!')

Furthermore, you can use a button to display this alert box using the button's onclick event. To do this, you need to define a function to be called:

function greet() {
  alert('Hello World!')
}

And then assign the function to the button's onclick event:

<button onclick="greet()"> Greet the world </button>
