Day 100 - JavaScript Objects - Pt. 5

Order in Objects

Object properties are semi-ordered: integer properties are sorted, while the rest appear in order of creation.

For instance, integer keys will be ordered:

let objInt = {
 2: 'two',
 3: 'three',
 1: 'one',
};

for (let key in objInt) {
 console.log(key); // 1, 2, 3
 console.log(objInt[key]); // one, two, three
}

But non-integer keys will appear in the same order as the creation:

let objStr = {
 two: 2,
 three: 3,
 one: 1,
};

for (let key in objStr) {
 console.log(key); // two, three, one
 console.log(objStr[key]); // 2, 3, 1
}

You can use a combination of both; the integers keys will appear sorted at the top, and then the rest of the keys in creation order:

let objMix = {
 three: 3,
 two: 2,
 1: 'one',
 0: 'zero',
};

for (let key in objMix) {
 console.log(key); // 0, 1, three, two
 console.log(objMix[key]); // zero, one, 3, 2
}
