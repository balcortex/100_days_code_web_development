Day 091 - Project: Shopping list - Pt. 3

There are two features to add to the shopping list webpage: the ability to mark items as completed and delete items that are no longer required.

Let's start with the easiest one, which is to delete items. For this matter, I chose to include a button on each item, created when the item is added to the list. The event listener is straightforward; we get the event's target, next get the parent, and finally remove the parent (which will delete the text and the button itself).

const delBtn = document.createElement('button');
delBtn.textContent = 'Delete';
delBtn.addEventListener('click', e => e.target.parentNode.remove())

Regarding marking the items as completed, I chose a checkbox (for now, as I wouldn't say I like the result very much). Like the delete button, we add an event listener to look for changes in the checkbox. As we are toggling classes, we don't need to check whether the checkbox is checked or not:

const checkBox = document.createElement('input');
checkBox.type = 'checkbox';
checkBox.addEventListener('change', e => e.target.parentNode.classList.toggle('items-done'));

The CSS rule to identify the item as `done` sets a line through the text and changes its color to a dark gray:

.items-done{
 text-decoration: line-through;
 color: #7c7c7c;
}
