Mixing Inline, Internal and External CSS

You can mix Inline, Internal, and External CSS Style with the following resolution order, ordered from most to least important:

  1. Inline CSS
  2. Internal CSS
  3. External CSS

If the three styles have conflicting attributes, the Inline CSS Style will be applied. If only the Internal CSS and External CSS have common attributes, the Internal CSS Style will be applied.

In this way, you can overrule a specific attribute as needed.
