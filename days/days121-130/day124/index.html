Line height

You can adjust the space between the lines in wrapped text using the line-height property:

<div class="container">
 <div>
  Lorem ipsum dolor...
 </div>
 <div style="line-height: 0.9em;">
  Lorem ipsum dolor...
 </div>
 <div style="line-height: 1.5em;">
  Lorem ipsum dolor...
 </div>
</div>

The result of the previous code is something like:

Narrow (0.9em)

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Numquam nesciunt maiores quisquam? Reprehenderit incidunt veniam suscipit debitis, mollitia ipsa deserunt dicta libero similique temporibus ratione nesciunt consectetur impedit blanditiis magni?
Wide (1.5em)

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Molestias pariatur ipsum magni obcaecati! Nihil earum magni autem, deserunt, minus illum dolor deleniti eaque sint, fuga alias tempora odit quas eius?

Text transform

You can change the case of any text by using the text-transform property. For instance, you can force a given text to be uppercase only:

<div style="text-transform: uppercase;">
 this text is written in lowercase in the HTML document
</div>

Which will show everything in uppercase:

Text shadow

The property text-shadow adds a shadow around the text in the selected element. This one is best used sparingly, but can be used to great effect in headings or other presentational text:

<div style="text-shadow: 2px 2px 7px black;">
 This text has a shadow
</div>

The previous code adds a shadow 2px off the text in both directions (x-axis and y-axis) with a blur of 7px and color black:
