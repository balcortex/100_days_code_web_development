Day 123 - CSS: Advanced Font Style - Pt. 2

Font style

We already saw that you can use the <em> tag to emphasize the text by italicizing its font. However, you should only use this tag (and the <strong>) whenever you want to give a semantic meaning rather than just styling.

If you only want to style the text, you should use the font-style attribute:

<div style="font-style: italic;">
 This text is in italics
</div>

Which is displayed as (aside from the background color):

This text is in italics

Letter spacing

You can adjust the space between the letters using the letter-spacing attribute. However, this should be used sparingly and with care. To make the space narrower, you can use negative units:

<div style="letter-spacing: -0.08em;">
 This text is narrow
</div>

Which results in:

This text is narrow

Similarly, you can increase the space using positive units:

<div style="letter-spacing: 1em;">
 This text is wide
</div>

Which yields:

This text is wide
