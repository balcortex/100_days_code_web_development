Day 121 - CSS Units Pt. 5

Percentage unit

This unit is similar to the `em` unit because both are relative to a parent's property. The difference is that the percentage unit could refer to different properties, while the `em` relates only to the font size.

For example, if you set an element's font-size as a percentage, it will be a percentage of the font-size of the element's parent. If you use a percentage for a width value, it will be a percentage of the parent's width.

Let's see the percentage unit in action making a div container half the size of the page (i.e., 50% of the parent) and the font twice as big (200%):

<div style="width: 50%; font-size: 200%;">

</div>

The same result could be achieved using the `vw` unit to specify the width and the `em` (or the `rem`) unit to set the font-size:

<div style="width: 50vw; font-size: 2em;">

</div>
