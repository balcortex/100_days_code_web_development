Day 122 - CSS: Advanced Font Style - Pt. 1

Fallback fonts

If you use the font-family property to change the font, and this font does not happen to be installed on your user's computer, then the default HTML font will be used. If you prefer to employ your own fallback font, you can use a list of fonts ordered from the most to the least preferred one. The browser will go through each font in the list until it finds the first installed (or again, it will use the default if no one is found):

.fallback-font {
 font-family: 'A non-existing font', 'Comic Sans MS', 'Other Fonts...';
}

As the first font does not exist in the system, the browser will use the Comic Sans MS font:

This text uses the Comic Sans MS font

Online fonts

An easy method to get fonts not installed on a user's computer is to use an online font library like Google Fonts. For instance, you can utilize the Roboto font by including the following import on top of your CSS document:

@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');

And use it directly on the font-family property:

.roboto-font {
 font-family: 'Roboto';
}

The browser then will download and display the font:

This is the Roboto font

Downloaded fonts

It is also possible to use a font you downloaded from the web. In your CSS file, you import and define a custom font using the @font-face rule:

@font-face {
 font-family: 'Miracle Dream';
 src: url(fonts/miracle_dream/Miracle\ Dream.otf);
}

And then use it as you would any other font-family:

.miracle-font {
 font-family: 'Miracle Dream';
}
