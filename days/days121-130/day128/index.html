Day 128 - CSS: Advanced Selectors - Pt. 3

Pseudo-elements

While pseudo-classes give us an alternative way to interact with our HTML elements based on their state or structure, pseudo-elements are more abstract. They allow us to affect parts of our HTML that aren't elements at all.

For instance, all the paragraphs in this document start with a red letter, which is easily achieved with the ::first-letter selector:

p::first-letter {
 color: red;
}

You can also customize the highlighted text using the ::selection selector. For instance, you can change the background and the color:

div::selection {
 color: yellow;
 background-color: red;
}

This code yields the following result while highlighting some words in the middle line (the purple background is only to distinguish the example from the actual content):

Lorem ipsum dolor sit amet consectetur adipisicing elit. Repellat deserunt minus eos, reprehenderit magni dicta deleniti placeat et aliquam dolorem quas velit nesciunt illo beatae quo possimus accusamus aperiam obcaecati.

Other useful pseudo-elements are the ::before and ::after. You can use both add content before and after the element:

.emphasis::before {
 content: "---";
}

.emphasis::after {
 content: "---";
}

Now, you can use the class emphasis to add three hyphens before and after automatically to any element:

<div class="emphasis"> This text is important </div>

Which is rendered by the browser like this:

This text is important

Note: the content inside ::before and ::after isn't accessible (i.e., the user cannot select it), so you should only use it for decorative purposes and not to add actual content.
