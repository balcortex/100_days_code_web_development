Day 127 - CSS: Advanced Selectors - Pt. 2

Pseudo-classes

A pseudo-class is a selector that selects elements in a specific state, e.g., they are the first element of their type or being hovered over by the mouse pointer.

Pseudo-classes are keywords that start with a colon. For example, :hover is a pseudo-class.

Dynamic pseudo-classes

These types of pseudo-classes help make the page more dynamic and interactive. For instance, you can apply a different style to any element that is under the pointer using the :hover pseudo-class:

div:hover {
 background-color: aqua;
}

The result is the following:

This div does not have a hover effect.
This div has a hover effect.
This div does not have a hover effect.

There are many dynamic pseudo-classes, like :focus, :active, :link, :visited.

Structural pseudo-classes

These pseudo-classes select elements based on their position within the DOM.

  • :root - a class that represents the top level of your document. It is generally where global CSS rules (i.e., rules you want to apply everywhere) are defined.
  • :first-child - will match only the first sibling.
  • :last-child - will match only the very last sibling.
  • :empty - will select elements that have no children.
  • :only-child - will match elements that have no siblings.

The :nth-child offers a more flexible approach:

  • .myList:nth-child(5) - selects the fifth element with class myList
  • .myList:nth-child(3n + 4) - selects every third element with class myList beginning at element four
  • .myList:nth-child(even) - selects every even element with class myList
  • .myList:nth-child(odd) - selects every odd element with class myList
