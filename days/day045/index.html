JavaScript: Hello World! Pt. 2

The second, more advanced way to make a "Hello World!" program in JavaScript is using the document.querySelector() and the textContent property in the returned element.

The document.querySelector() returns the first element within the document that matches the specified selector (or group of selectors). If no matches are found, null is returned.

On the other hand, the textContent property of an element represents the text content of the element and its descendants.

Suppose we have an h2 element displaying the text: "This is the original text." To dynamically change its content, we can insert the following code in the HTML after declaring the h2 element (if you put the code before the h2 tag, the text set by JavaScript will be overwritten by the h2):

const myHeading = document.querySelector("h2");
myHeading.textContent = "Hello world!";

Remember that the code must be within a script tag.

This is the original text

As soon as we load the page, the text will change from "This is the original text" to "Hello World!". This change happens instantly, so you will no be able to see the original text.

In this manner, you can dynamically change any document text using JavaScript.
