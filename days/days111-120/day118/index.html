Day 118 - CSS Units Pt. 2

Relative units

Relative units are units that can change based on their context. As their name suggests, these units are relative to something else, perhaps the size of the parent element's font, or the size of the viewport.

The benefit of using relative units is that you can make the size of text or other elements scales relative to everything else on the page. That is, the size of an element will adjust whether the user has a big or small screen (it does not depend on the resolution, like the pixel for example).

There are several relative units, like `em`, `rem`, `vw`, `vh`, etc. Let's start with the `em` unit.

em unit

`em` refers to the font size, though it could be used to define other attributes.

`1em` is the font size of the element's parent, in the case of typographical properties like font-size, and the font size of the element itself in the case of other properties like width.

For instance,

Parent (16px)
Child (1em = 16px)
Grandchild (1em = 16px)

Parent (10px)
Child (2em = 2*10px = 20px)
Grandchild (2em = 2*20px = 40px)

Parent (16px)
Child (0.5em = 0.5*16px = 8px)
Grandchild (2em = 2*8px = 16px)
