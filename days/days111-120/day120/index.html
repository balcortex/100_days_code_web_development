Day 120 - CSS Units Pt. 4

Viewport units

These units relate to the viewport's size (i.e., the visible portion of the document), which can be helpful any time you want something to be sized relative to the viewport. Examples include full-height heroes and full-screen app-like interfaces.

There is a viewport-percentage length unit variant corresponding to each of the viewport sizes:

vh

Represents a percentage of the height of the viewport's initial containing block. 1vh is 1% of the viewport height.

vw

Represents a percentage of the width of the viewport's initial containing block. 1vw is 1% of the viewport width

vmax

Represents in percentage the largest of vw and vh.

vmin

Represents in percentage the smallest of vw and vh.

These units are mainly used to define the width and height of the elements. For example, you can make a div element half the width of the viewport width:

<div style="width: 50vw;">
 This div is half the width of the viewport
</div>

Which looks something like this:

This div is half the width of the viewport

Or, you can set its height to be 10% of the viewport's height:

<div style="height: 10vh;">
 This div is 10% the height of the viewport's height
</div>

Which gives the following result:

This div is 10% the height of the viewport's height
