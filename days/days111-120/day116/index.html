Day 116 - HTML Tables, Pt. 5

If your table needs a caption (and it should), you can give it one with the <caption> element. You should put it just below the opening <table> tag:

<table>
 <caption>
  My 2024 School Timetable
 </caption>
 ...

The caption is placed directly above the table:

My 2024 School Timetable
Mon Tues Wed Thurs Fri
09:00 Math Chemistry Science Chemistry Biology
10:00 Biology Arts Arts
11:00 Break
12:00 Arts Biology Science Math Arts
Art eqpt. Yes No Yes No Yes

You can give your table a more structured definition using the elements <thead>, <tfoot>, and <tbody>, which allows you to mark up a header, footer, and body section for the table. These elements are helpful for styling but do not provide any style by default. Let's use them alongside the following CSS rules to improve the visual aspect of the table further:

thead {
 text-transform: uppercase;
}

tbody {
 font-family: "Lucida Handwriting", cursive;
}

tfoot {
 text-align: center;
 font-size: 12px;
 background-color: rgba(79, 96, 255, 0.338);
}

The result is rendered as follows:

My 2024 School Timetable
Mon Tues Wed Thurs Fri
09:00 Math Chemistry Science Chemistry Biology
10:00 Biology Arts Arts
11:00 Break
12:00 Arts Biology Science Math Arts
Art eqpt. Yes No Yes No Yes

Note: <tbody> is always included in every table, implicitly if you don't specify in your code.
