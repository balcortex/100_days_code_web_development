Day 114 - HTML Tables, Pt. 3

We've seen that adding style to a row (or set of rows) is straightforward: you just have to add it directly to the `tr` tag. But what about the columns? Remember that a column in an HTML table is not directly defined as a row; cells are added as needed to make up for the columns. Said this, the first approach would be to add the style directly in each cell of the column:

<table>
 ...
 <tr>
  <td style="background-color: yellowgreen;">09:00</td>
  <td>Math</td>
  ...
 <tr>
  <td style="background-color: yellowgreen;">10:00</td>
  <td>Math</td>
  ...

We added a green background to each of the cells in the first column, which results in:

Mon Tues Wed Thurs Fri
09:00 Math Biology Science Chemistry Arts
10:00 Math Biology Science Chemistry Arts
11:00 Break Break Break Break Break
12:00 Math Biology Science Chemistry Arts

For this table, we only had to modify five cells. But, of course, if the table had thousands of rows, this wouldn't be feasible. Fortunately, there's a way to define a column (or set of columns) using the `colgroup` element. The equivalent of the previous code, using this approach, is:

<table>
 <colgroup>
  <col style="background-color: yellowgreen;">
 </colgroup>
 <tr style="background-color: darkgrey;">
 ...

Which results in the following table:

Mon Tues Wed Thurs Fri
09:00 Math Biology Science Chemistry Arts
10:00 Math Biology Science Chemistry Arts
11:00 Break Break Break Break Break
12:00 Math Biology Science Chemistry Arts

However, this approach has a catch: the background color is only applied if the cell has no other background, which can be helpful or not so much, depending on the situation.

Use the' span' property to apply the same style to contiguous columns. For instance, let's make the first two green, the next two with no color, and the last two blue and red, respectively:

<table>
 <colgroup>
  <col style="background-color: yellowgreen;" span="2">
  <col span="2">
  <col style="background-color: aqua;">
  <col style="background-color: orangered;">
 </colgroup>
 <tr style="background-color: darkgrey;">
 ...

Which is rendered as:

Mon Tues Wed Thurs Fri
09:00 Math Biology Science Chemistry Arts
10:00 Math Biology Science Chemistry Arts
11:00 Break Break Break Break Break
12:00 Math Biology Science Chemistry Arts
