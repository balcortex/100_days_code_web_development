Day 113 - HTML Tables, Pt. 2

We already covered the basic elements of HTML tables, let's dig a bit deeper. First, we define a table containing a school timetable:

<table>
 <tr>
  <td></td>
  <td>Mon</td>
  <td>Tues</td>
  <td>Wed</td>
  <td>Thurs</td>
  <td>Fri</td>
 </tr>
 <tr>
  <td>09:00</td>
  <td>Math</td>
  ...

Which is rendered something as (depending on the style you have already defined):

Mon Tues Wed Thurs Fri
09:00 Math Biology Science Chemistry Arts
10:00 Math Biology Science Chemistry Arts
11:00 Break Break Break Break Break
12:00 Math Biology Science Chemistry Arts
13:00 Math Biology Science Chemistry Arts

The first issue that pops out is that all cells have the same style, and at first glance it's not so easy to see the important reference cells (table headers). Let's make the row heading stood out, using the `th` tag (table header) instead of the `td` tag:

<table>
 <tr>
  <th></th>
  <th>Mon</th>
Mon Tues Wed Thurs Fri
09:00 Math Biology Science Chemistry Arts
10:00 Math Biology Science Chemistry Arts
11:00 Break Break Break Break Break
12:00 Math Biology Science Chemistry Arts
13:00 Math Biology Science Chemistry Arts

The table looks better, and the header stands out easily. By default, table headings have some default styling: bold and centered text. There are other benefits of using table headers, for instance, screen readers can associate a whole column with its header, and read out all the data from this column, instead of going throughout the whole table.
