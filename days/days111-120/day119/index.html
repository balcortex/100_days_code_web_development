Day 119 - CSS Units Pt. 3

rem unit

As the `em` unit, the `rem` unit refers primarily to the font size of an element. Still, instead of the parent's size, it relates to the root element's size.

The math works the same as with the `em` unit: `1rem` is the font size of the root element (the html element most of the time), and `2rem` is twice the size of the root element font size.

Let's compare both the `em` and the `rem` units with three nested divs, as the following code shows:

<div>
 Parent
 <div>
  Child
  <div>
   Grandchild
  </div>
 </div>
</div>

Let's start by giving all the div elements the class of `div-em`:

.div-em{
 font-size: 1.5em;
}

This results in an increase of 50% the font size in each nested level. The further the nesting, the bigger the font size:

Parent
Child
Grandchild

Now, let's try with the `div-rem` class:

.div-rem{
 font-size: 1.5rem;
}

Contrary to the last example, all the divs have the same font size, which is 50% bigger than the root element font size. Since the root element is the html element, with a default size of 16px, all the divs will have a font size of 24px:

Parent
Child
Grandchild

Most of the time, you want all the elements to refer to the root element. Thus, the `rem` unit is often more used.
